# Tree

**Tree** shows you directories arbitrary levels deep.

## Install

With [Fisherman]

```
fisher install jethrokuan/tree
```

## Usage

```fish
t : tree 1 level deep
t2: tree 2 levels deep
...
t6: tree 6 levels deep
```
